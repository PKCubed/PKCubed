# Hello! I'm Peter Kyle.

**I'm a junior Electrical Engineering student at Walla Walla University based in Wenatchee, WA.** I have a multitude of hobbies, but one of my favorite things to do is just build things and I share most of my projects here.

## Current Projects
[Software Defined Radio](https://github.com/SomethingFourier/SDR-Receiver), a hybrid superheterodyne direct conversion quadrature software defined radio receiver supporting a wide frequency range from HF to VHF bands, 192kHz of active bandwidth, both USB and Ethernet data transfer, and Power over Ethernet.

[OpenFader](https://github.com/PKCubed/OpenFader), a backplane PCB for motorized faders (linear potentiometers) offering easy, modular, serialized UART control.

[LibreCom](https://github.com/PKCubed/LibreCom), an open source and affordable wireless voice communication system for live events and other demanding environments.

[MQTT Button Panel](https://github.com/PKCubed/MQTT-Button-Panel), a panel of 4 programmable buttons designed to be used with Home Assistant featuring capacitve touch, versitile button modes, and a comprehensive web interface for configuration.

[OpenChime](https://github.com/PKCubed/OpenChime), an open source school bell system solution designed to be installed on a raspberry pi offering scheduling with Google Calendar, multiple bell sounds, and more.
